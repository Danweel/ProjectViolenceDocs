Installing
~~~~~~~~~~

    1. Open a text editor window and open the script file. You can just drag the py script file directly into the window.
    2. Press the play button at the top of the window.

You don't need to do this on files that are already set up by Fred, like an inbetween or test file.

However, when a revision comes out, you may want to update.

.. note:: 'Playing' the script file will **rewrite** the changes you may have made to it if any, you have to save first.
